# Facebook Comment Automation for Railway

لوحة تحكم عربية لأتمتة تعليقات صفحة فيسبوك على `Railway`.

## الميزات

- الرد التلقائي على تعليقات المنشورات فقط.
- الإعجاب التلقائي بتعليقات المنشورات فقط.
- تجاهل ردود التعليقات بالكامل.
- اختيار `المنشورات الجديدة` أو `كل المنشورات` للرد.
- اختيار `تعليقات المنشورات الجديدة` أو `تعليقات حتى المنشورات القديمة` للإعجاب.
- تحديد زمن الانتظار بين كل رد والذي يليه.
- تحديد زمن الانتظار بين كل إعجاب والذي يليه.
- سجل عمليات وآخر التعليقات التي تم اكتشافها.

## المتغيرات المطلوبة

```env
PAGE_ACCESS_TOKEN=...
PAGE_ID=108133832082057
VERIFY_TOKEN=...
DASHBOARD_PASSWORD=...
```

## إعداد Meta Webhook

- `Callback URL`: `https://YOUR-DOMAIN.up.railway.app/webhook`
- `Verify Token`: نفس قيمة `VERIFY_TOKEN`
- `Webhook Field`: `feed`

## ملاحظة مهمة

التوكن الحالي في بيئة العمل انتهت صلاحيته، لذلك يجب توليد `Page Access Token` جديد قبل أن يبدأ الرد أو الإعجاب فعليًا.
